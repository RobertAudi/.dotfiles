Rust
----

### Install/Update

Use the `setup` script to install Rust using [rustup.rs][]:

```console
$ setup rust
```

Use the `setup` script to install/update default cargo packages:

```console
$ setup cargo-packages
```

[rustup.rs]: https://rustup.rs/
