This repository contains my dotfiles managed by [GNU Stow][stow].

Requirements
------------

- macOS
- tmux 2.4+
- zsh 5.4+
- Vim/MacVim
- [fasd][fasd]

[stow]: https://www.gnu.org/software/stow/
[fasd]: https://github.com/clvv/fasd

Bootstrapping
-------------

Perl: install the `local::lib` module:

```sh
cpan install local::lib
```
