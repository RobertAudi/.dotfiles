Vimium key mappings
===================

Navigating the page
-------------------

```
scrollDown                                         j, <c-e>         Scroll down
scrollUp                                           k, <c-y>         Scroll up
scrollToTop                                        gg               Scroll to the top of the page
scrollToBottom                                     G                Scroll to the bottom of the page
scrollPageDown                                     d                Scroll a half page down
scrollPageUp                                       u                Scroll a half page up
scrollFullPageDown                                 <Nop>            Scroll a full page down
scrollFullPageUp                                   <Nop>            Scroll a full page up
scrollLeft                                         h                Scroll left
scrollRight                                        l                Scroll right
scrollToLeft                                       zH               Scroll all the way to the left
scrollToRight                                      zL               Scroll all the way to the right
reload                                             r                Reload the page
copyCurrentUrl                                     yy               Copy the current URL to the clipboard
openCopiedUrlInCurrentTab                          p                Open the clipboard's URL in the current tab
openCopiedUrlInNewTab                              P                Open the clipboard's URL in a new tab
goUp                                               gu               Go up the URL hierarchy
goToRoot                                           gU               Go to root of current URL hierarchy
enterInsertMode                                    i                Enter insert mode
enterVisualMode                                    v                Enter visual mode
enterVisualLineMode                                V                Enter visual line mode
passNextKey                                        <Nop>            Pass the next key to the page
focusInput                                         gi               Focus the first text input on the page
LinkHints.activateMode                             f                Open a link in the current tab
LinkHints.activateModeToOpenInNewTab               F                Open a link in a new tab
LinkHints.activateModeToOpenInNewForegroundTab     <Nop>            Open a link in a new tab & switch to it
LinkHints.activateModeWithQueue                    <a-f>            Open multiple links in a new tab
LinkHints.activateModeToDownloadLink               <Nop>            Download link url
LinkHints.activateModeToOpenIncognito              <Nop>            Open a link in incognito window
LinkHints.activateModeToCopyLinkUrl                yf               Copy a link URL to the clipboard
goPrevious                                         [[               Follow the link labeled previous or <
goNext                                             ]]               Follow the link labeled next or >
nextFrame                                          gf               Select the next frame on the page
mainFrame                                          gF               Select the page's main/top frame
Marks.activateCreateMode                           m                Create a new mark
Marks.activateGotoMode                             `                Go to a mark
```

Using the vomnibar
------------------

```
Vomnibar.activate                                  o                Open URL, bookmark or history entry
Vomnibar.activateInNewTab                          O                Open URL, bookmark or history entry in a new tab
Vomnibar.activateBookmarks                         b                Open a bookmark
Vomnibar.activateBookmarksInNewTab                 B                Open a bookmark in a new tab
Vomnibar.activateTabSelection                      T                Search through your open tabs
Vomnibar.activateEditUrl                           ge               Edit the current URL
Vomnibar.activateEditUrlInNewTab                   gE               Edit the current URL and open in a new tab
```

Using find
----------

```
enterFindMode                                      /                Enter find mode
performFind                                        n                Cycle forward to the next find match
performBackwardsFind                               N                Cycle backward to the previous find match
```

Navigating history
------------------

```
goBack                                             H                Go back in history
goForward                                          L                Go forward in history
```

Manipulating tabs
-----------------

```
createTab                                          t                Create new tab
previousTab                                        J, gT            Go one tab left
nextTab                                            K, gt            Go one tab right
visitPreviousTab                                   ^                Go to previously-visited tab
firstTab                                           g0               Go to the first tab
lastTab                                            g$               Go to the last tab
duplicateTab                                       yt               Duplicate current tab
togglePinTab                                       <a-p>            Pin or unpin current tab
toggleMuteTab                                      <a-m>            Mute or unmute current tab
removeTab                                          x                Close current tab
restoreTab                                         X                Restore closed tab
moveTabToNewWindow                                 W                Move tab to new window
closeTabsOnLeft                                    <Nop>            Close tabs on the left
closeTabsOnRight                                   <Nop>            Close tabs on the right
closeOtherTabs                                     <Nop>            Close all other tabs
moveTabLeft                                        <<               Move tab to the left
moveTabRight                                       >>               Move tab to the right
```

Miscellaneous
-------------

```
showHelp                                           ?                Show help
toggleViewSource                                   gs               View page source
```
